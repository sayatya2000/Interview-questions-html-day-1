HTML INTERVIEW QUESTIONS DAY 1

1.Difference between div and span.


div tag or divition tag is a block level element.It takes the whole width of the webpage.


Span tag is an inline element. It only takes spaces as per the elements inside of it.


2.Difference between class and ID.


A class name can be used by multiple HTML elements, while an ID name must only be used by one HTML element with in the page.


3.What are self closing tags?


Self closing tag are tag thats do not need a closing tag. eg: br tag, hr tag, image tag and meta tag etc.


4.How many types of headings are there?How to decide which to use where


There is 6 type of headings. Headings are H1, H2, H3, H4, H5 and H6.


  • H1 tags are used to denote the most important text, suchas the main theme or title of a content.
  • H2 and H3 are commonly used as subheadings.
  • Finally, H4, H5 and H6 tags may be used to provide further structure within those subsections.

5.Difference between block and inline.


Block level elements takes the whole width of the webpage and Inline element only takes places as per the elements presents inside of it.


example of block level element is heading tag, lists and paragraph tag etc. and example of inline element is anchor tag, strong tag, bold tag and italic tag etc.


6.Difference between em and strong.


emphasis tag converts the text into italic style. and strong tag highlight the particular text into bold style.


7.What are attributes?


Attribute are some special words that provides additional info about the specific element that you have made. eg. action, method, src and herf etc.
